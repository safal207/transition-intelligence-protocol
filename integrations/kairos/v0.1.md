# TIP → Kairos Integration Profile v0.1

## Purpose

This profile exports a bounded TIP transition proposal to Kairos Gate for research-only phase-window assessment.

```text
IFP Ready State
→ TIP justified transition
→ TIP export profile
→ Kairos phase-window assessment
```

It is an interface contract, not a third protocol and not a merger of TIP with Kairos.

## Required TIP export

A conforming export contains:

- exact starting-state reference;
- tension;
- cause;
- proposed transition;
- smallest justified research step;
- one or more evidence references;
- producer, timestamp, and content digest;
- exact TIP commit reference.

The machine-readable required-field profile is `profile.v0.1.json` in this directory.

## Kairos extension

Kairos adds a supported pre-intervention phase variable, forecast horizon, identity-preservation proxy, toxicity or stress proxy, reversibility or recovery proxy, evidence quality, timing confidence, and a research-only classification.

## Authority boundary

TIP exports context; it does not authorize Kairos output. Kairos classifies a research candidate window; it does not authorize biological execution.

Neither protocol grants wet-lab, animal, human, clinical, deployment, approval, delivery, ownership, or merge authority.

## Compatibility rule

A TIP export is eligible for the Kairos handoff only when every required field in `profile.v0.1.json` is present and non-empty, evidence references are non-empty, provenance is complete, and the exact TIP commit is recorded.

The full cross-repository JSON Schema and deterministic validator live in Kairos Gate to avoid two diverging copies of the same receiving contract.
